## 1) Input Validation (Home Page)

## 3) Default Behavior (Edge Case)

* Populate "other" with:
  ["Communication", "Problem solving", "Basic coding", "Projects"]

## 4) Score Stability Rules

* baseScore is calculated ONLY during analysis
* finalScore updates ONLY when skillConfidenceMap changes

  * Update updatedAt
